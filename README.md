# gender-bias dataset
Dataset resulting from the study “Detection of gender-based violence in digital environments using AI models”
It includes comments on the teaching survey annotated as gender-bias or not-gender-bias 

It is composed of:
dataset960_gender_vias: 960 rows 
trainext_gender_bias: 770 rows (80.2%)
testext_gender_bias: 190 rows (19.8%)
